USCIS case status check

If you are applying for a 🇺🇸US visa you might find yourself constantly checking the USCIS website for updates - which can be quite nerve-wracking. I searched for USCIS scrapers that could automatically notify me of any changes, but the ones I came across were outdated and no longer functional (at least didn’t work for me). I decided to give it a try with Selenium, GitHub Actions and SendGrid and ended up with a script that checks my status and sends me an email if anything changes.

Tools: Selenium GitHub Actions, Sendgrid

Lumen API Search Tool

Lumen is an independent research project managed by the Berkman Kein Center for Internet & Society at Harvard aw School. It publishes copies of online content takedown requests shared with Lumen by Google. This tool leverages the Lumen API to automate searches to study and identify networks of interconnected - likely fraudulent - takedown requests. It results of a previous search as a search term for a new search.

Tools: API, Python
